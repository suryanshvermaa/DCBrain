# UI Guidelines

## Design Principles
- Modern, clean, and highly professional.
- Focus on data density without clutter.

## Colors & Typography
- **Primary:** Deep Blue, Tech Green, Alert Red.
- **Font:** Inter (Sans-serif) for dashboards.

## Components & Layout
- Use Shadcn/UI for React components.
- Dashboard: Sidebar navigation, top bar for search/profile, main content area for widgets.

## Visualizations
- **Knowledge Graph Visualizer:** Use specialized graph visualization libraries (e.g., React Flow or Cytoscape) to display Failure Propagation and Document Relationships.
- **Timeline:** Interactive Gantt charts for simulation delay visualization.

## Responsive Design
- Mobile-first approach for progressive web app (future scope).
- Data tables must scroll horizontally on small screens.
