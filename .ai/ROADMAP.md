# Roadmap

## Phases
**Phase 1: Foundation (Q3 2026)**
- Setup, Auth, Document Upload (MinIO), Basic RAG Search, Chat, Basic Dashboard.

**Phase 2: Intelligence & Knowledge Graph (Q4 2026)**
- Compliance Validation Engine, Schedule Risk Prediction, Procurement Dashboards.
- Knowledge Graph integration (Equipment/Vendor/Document relationships).
- Release of core AI Agents (Document, Compliance, Knowledge).

**Phase 3: Scale & Simulation (Q1 2027)**
- Supervisor Agent orchestration.
- Delay Simulation & What-if Simulation Engine.
- Rollout of remaining Agents (Mitigation Planner, Commissioning, Executive Copilot, Data Validation, etc.).
- Multi-project support, Integration Hub.

## Dependencies
- Document Processing -> RAG Search -> Knowledge Agent.
- MinIO Storage -> Document Processing Pipeline.
- Knowledge Graph -> Supervisor Agent Context.
- Schedule Import -> Risk Analysis -> Simulation Engine.
